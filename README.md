# Top-Down - Grupo 8

This is a top-down game created for the Activity 3 from the Group 8



## 游 Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

* [Unity3D v6000.2.7f2](https://unity.com/releases/editor/archive)

### Installation

1. Clone or download the repository.
2. Open your Unity3d Editor
3. Click **Add** and select the root folder of this project.
4. Open the project and wait for the libraries to import.
